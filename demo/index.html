A Better ScrollTo Plugin

By Me (HipsterBrown)

Jump to TL;DR

I started creating something like this in a project at work. I was tired of seeing $.animate({scrollTo}) when I knew how bad it was for performance, especially because jQuery.animate doesn't use requestAnimationFrame. I also wanted a chance to try creating a JavaScript plugin with a simple API and great performance.

'ScrollTo' animations are pretty slick when implemented properly. Rather than just linking to an element's id on the page and letting the brower jump-cut to the element, scrollTo animations can give the user a smoother transition to that element up or down on the page. But it is important to keep accessibility and fallbacks in mind. In the original version of this plugin, I used the window.pushState API to update the URL of the page, so if the page is refreshed the scroll position is saved along with an accurate browsing history.
